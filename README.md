No backend. All filtering client-side. Saves stored in localStorage.

## Quickstart
1) `npm i`
2) `npm run dev`
3) open http://localhost:3000

## Deploy (Vercel)
- `npm i -g vercel`
- `vercel`

## Customise
- Data: edit `/public/data/venues.json` (you can add `image` later)
- Colors: `tailwind.config.ts`
- Styles: `styles/globals.css`
- Scoring: edit `app/page.tsx` in the `results` computation.
